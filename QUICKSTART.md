# PewTrack Frontend Demo - Quick Start

## Deploy to Vercel (Recommended)

1. **Connect Repository**
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project"
   - Import your GitHub repository

2. **Deploy**
   - Vercel will automatically detect Next.js
   - No configuration needed
   - Deploy will complete in ~2 minutes

3. **Access Your Demo**
   - Visit the provided Vercel URL
   - Start exploring the PewTrack interface

## What You'll See

- **Dashboard** (`/`): Overview of test sessions
- **New Test** (`/new`): Configure firearm testing parameters
- **Active Test** (`/test/1`): Interactive firing grid with stoppage tracking
- **Measurements** (`/measurements/1`): Maintenance and measurement history

## Demo Features

✅ Create test configurations

✅ Stoppage logging

## No Backend Required

This demo runs entirely in the browser using:
- Mock data for immediate testing
- localStorage for state persistence
- Static Next.js build for Vercel
